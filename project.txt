# 🧹 Step 2 — Text Preprocessing (Foundation)

### 📌 Why this matters:

Raw text is messy → ML models need **clean, consistent input**

### Create `utils.py`

```python
import re

def clean_text(text):
    text = text.lower()
    text = re.sub(r'[^a-zA-Z0-9\s]', '', text)
    text = re.sub(r'\s+', ' ', text).strip()
    return text
```

### 🧠 Concept:

* Normalization = improves similarity accuracy
* Removes noise → better embeddings

---

# 🔢 Step 3 — Convert Text to Vectors (Core ML Step)

### 📌 Why:

Machines don’t understand text → convert to **numerical vectors**

### Create `model.py`

```python
from sentence_transformers import SentenceTransformer

model = SentenceTransformer('all-MiniLM-L6-v2')

def get_embedding(text):
    return model.encode(text)
```

### 🧠 Concept:

* Embeddings capture **semantic meaning**
* “Python developer” ≈ “software engineer (Python)” → close vectors

---

# 📏 Step 4 — Similarity Scoring

### 📌 Why:

We need a **match score (0–100%)**

Add to `model.py`:

```python
from sklearn.metrics.pairwise import cosine_similarity

def compute_similarity(resume_emb, jd_emb):
    score = cosine_similarity([resume_emb], [jd_emb])[0][0]
    return round(score * 100, 2)
```

### 🧠 Concept:

* Cosine similarity measures **angle between vectors**
* Closer angle → higher similarity → better match

---

# 🔍 Step 5 — Extract Skills (Basic Version)

### 📌 Why:

To give **actionable feedback**, not just score

Add to `utils.py`:

```python
SKILLS = ["python", "sql", "machine learning", "data analysis", "deep learning"]

def extract_skills(text):
    found = []
    for skill in SKILLS:
        if skill in text.lower():
            found.append(skill)
    return found
```

---

# ⚖️ Step 6 — Skill Gap Analysis

### 📌 Why:

This is where your project becomes **useful**

Add:

```python
def skill_gap(resume_skills, jd_skills):
    missing = list(set(jd_skills) - set(resume_skills))
    return missing
```

### 🧠 Concept:

* Set difference = identifies **missing requirements**
* This is your **recommendation engine**

---

# 🧪 Step 7 — Combine Everything

### Create `main.py`

```python
from fastapi import FastAPI
from app.utils import clean_text, extract_skills, skill_gap
from app.model import get_embedding, compute_similarity

app = FastAPI()

@app.post("/analyze")
def analyze(resume: str, jd: str):
    # Clean
    resume_clean = clean_text(resume)
    jd_clean = clean_text(jd)

    # Embeddings
    resume_emb = get_embedding(resume_clean)
    jd_emb = get_embedding(jd_clean)

    # Score
    score = compute_similarity(resume_emb, jd_emb)

    # Skills
    resume_skills = extract_skills(resume_clean)
    jd_skills = extract_skills(jd_clean)

    missing = skill_gap(resume_skills, jd_skills)

    return {
        "match_score": score,
        "resume_skills": resume_skills,
        "missing_skills": missing
    }
```
